Bootstrap

Bootstrap is a free and open-source front-end web framework for designing websites and web applications. It contains HTML- and CSS-based design templates for typography, forms, buttons, navigation and other interface components, as well as optional JavaScript extensions. We used Bootstrap for the responsive website because we wanted to be sure the game could run on mobile devices

For the rendering of graphics we used the standard html canvas. Because the canvas was tested on every supported html5 browser I was sure it would work on mobile devices as well.
